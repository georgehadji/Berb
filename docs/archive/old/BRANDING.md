## 📝 Messaging Framework

### Elevator Pitch (30 seconds)
> "Berb is an AI-powered research automation platform that reduces research costs by 85% while improving quality by 32%. It handles everything from literature search to paper writing, with a self-improving system that gets smarter with every project."

### Value Propositions
1. **Cost Efficiency** — 85-92% cost reduction
2. **Quality Improvement** — +32% quality score
3. **Speed** — 3 hours → 1-1.5 hours per project
4. **Coverage** — 20-30 → 70-100 papers analyzed
5. **Self-Improving** — Continuous learning system

### Key Features
- 📚 Comprehensive literature search (6+ grey literature sources)
- 🤖 Multi-model AI routing (DeepQuery, DeepMind AI, Claude, GPT)
- ✍️ Full paper drafting with 2M token context
- 🔍 Citation verification with real-time web search
- 🧠 Self-improving through experience analysis
- 💰 85-92% cost optimization

## 🔤 Usage Guidelines

### DO ✅
- Use "Berb" as a proper noun (capitalized)
- Use Berb Blue for primary branding
- Keep logo clear and legible
- Maintain consistent spacing

### DON'T ❌
- Don't use lowercase "berb" (except in code)
- Don't stretch or distort the logo
- Don't use outdated colors
- Don't combine with other brand elements
